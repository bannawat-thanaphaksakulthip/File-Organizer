# File Organizer

A Python script that organizes files into folders based on their file type.

## Features
- Sort files by extension (.jpg, .pdf, .py)
- Create folders automatically
- Move files into corresponding folders
- Accept user input for target directory

## Tech
- Python

## How to run
Run the script using Python:

```bash
python organizer.py
```
Then enter the folder path
```bash
enter folder path : C:\Users\YourName\Desktop\test-folder
```

## Example
Before:
```bash
file1.jpg
file2.pdf
script.py
```

After:
```bash
jpg/
  file1.jpg
pdf/
  file2.pdf
py/
  script.py
```
